Google Summer of Code 2013


As a participant of Google Summer of Code 2013 with The Wiselib.

Wiselib is a generic algorithm library for heterogenous sensor networks.
In simpler words, it contains various algorithm classes (for instance, localization or routing) that can be compiled for several platforms such as iSense or Contiki, or the sensor network simulator Shawn. It is completely written in C++, and uses templates in the same way as Boost and CGAL. This makes it possible to write generic and platform independent code that is very efficiently compiled for the various platforms.

My task was to port Arduino to wiselib which involved providing interfaces for timer implementation and radio modules like Series 1 802.15.4 chip antennae, Series 2 Radio (Zigbee) and Bluetooth Radio.

To check out my contributions, you can view my fork of The Wiselib.
